# Password Generator

Ein Python Passwort-Generator mit wählbarer Länge und optionalen Sonderzeichen

##Funktion

- Passwort generieren
- Länge des Passworts auswählen
- Optional Sonderzeichen hinzufügen

##Nutzung 

Starte das Programm mit:

Python password_generator.py

Danach im Menü auswählen: 
Passwort Generieren,
Passwortlänge,
Sonderzeichen, 
Beenden  

## Beispiel

1 : Passwort Generieren
2 : Wählen sie länge des Passwortes
3 : Wollen sie ein Sonderzeichen?
4 : Beenden

Wähle eine Option: 2
Länge: 12

Wähle eine Option: 3
Wollen sie das Passwort mit sonderzeichen? (y/n): y

Wähle eine Option: 1
Dein Passwort: A7f$Kp!QzLm2
